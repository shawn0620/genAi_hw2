# GenAI HW02 v2 系統介紹

學號：113550160

## 核心功能

1. 長期記憶  
   可跨聊天室保存使用者 facts，並支援查看、部分刪除與更新。

2. 多模態  
   支援文字、圖片與 PDF 上傳，PDF 會先轉成文字再交給模型處理。

3. 模型自動路由  
   依圖片、文件與任務複雜度，自動選擇 fast、reasoning、vision 模型。

4. Tool use / MCP  
   提供 `get_time`、`calculate`、`memory_recall` 三個工具，並實作正式 `POST /mcp` JSON-RPC 端點。

5. 其他功能  
   支援聊天紀錄管理、模型顯示、手動選模、跨聊天室記憶與本機附件上傳。

## 系統流程

1. 使用者在前端輸入文字，或附加圖片 / PDF。
2. 前端將訊息與設定送到 `/api/chat`。
3. 後端先進行模型路由，並查詢相關長期記憶。
4. 若需要，模型可呼叫工具；若原生 tool calling 未使用，則退回本地 heuristic fallback。
5. 回答以 SSE 串流回傳前端，回合結束後更新短期記憶與長期記憶。

## 系統架構

`User -> Frontend UI -> Backend API -> Router / Short-term Memory / Long-term Memory / Tool Engine / MCP -> LLM Provider -> SSE stream back to Frontend`

## 特色與價值

- 能根據任務型態自動切換模型，兼顧速度、品質與多模態能力。
- 可記住使用者偏好與個人資料，讓跨對話互動更連續。
- 透過工具與 MCP 降低幻覺，展示較接近標準化 agent 系統的整合方式。
